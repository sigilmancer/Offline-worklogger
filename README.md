# WhatNext-Base: The Zero-Cloud Work Logger

WhatNext is meant for solo freelancers. It is a 100% offline, native desktop app built to eliminate SaaS anxiety and ensure total data sovereignty.

This is the **Base Edition** : a fully functional, unlimited work logger. There are no "trial periods" or artificial limits on your logs.

##  Quick Start
1. Download the ZIP from the latest Release.
2. Run the **WhatNext-Base.exe** (Portable; no installation required).
3. Start logging. Everything stays on your machine.

##  The Architecture of Privacy
Built with SvelteKit and Electron, WhatNext uses a local IndexedDB for persistence.

* **Persistence:** Unlike web-trackers, your data won't vanish if you clear your browser cache.
* **100% Private:** No phone-home telemetry. No account required.
* **Zero Footprint:** No registry clutter. Run it from a folder or USB.
* **Data Sovereignty:** Use the built-in JSON Export/Import for full data ownership.

##  Security & Trust
